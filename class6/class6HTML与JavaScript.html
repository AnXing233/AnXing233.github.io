JavaScript 使 HTML 页面更具动态性和交互性

实例：

				
<p>我的第一段JavaScript</p>
<button type="button"
onclick="document.getElementById('demo').innerHTML = Date()">
点击我来显示日期和时间
</button>
<p id="demo"></p>

效果：

我的第一段JavaScript

说明：

这里引入一个 <button> 标签，这个是用来定义一个可点击的按钮。

<button> 中的"onclick"属性，是指点击后的事件(俗称点击后会干说什么)

这里 onclick 属性里是属性值，就是一段JavaScript代码

我们分开来看：

document.getElementById('demo').innerHTML = Date()

		

在此文档(该网页)                                (document)


寻找一个名为“demo”的元素                        (getElementById('demo'))


在这个元素所在的标签更改HTML元素                  (innerHTML)


把显示日期的函数      (指Date())      的这个值赋予给了这个id
管你看没看懂，NB就完事了！[doge]

<script> 标签

HTML <script> 标签用于定义客户端脚本（JavaScript）。

<script> 元素即可包含脚本语句，也可通过 src 属性指向外部脚本文件。

JavaScript 的常见用途是图像处理、表单验证和内容的动态更改。

如需选取 HTML 元素，JavaScript 最常用 document.getElementById() 方法。


这里的例子是在有“idname”的这个id的标签更改里面的内容

实例：

			
<script>
	document.getElementById('idname').innerHTML = "你好JavaScript";
</script>
