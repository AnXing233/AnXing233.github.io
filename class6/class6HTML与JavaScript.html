<script> 标签

HTML <script> 标签用于定义客户端脚本（JavaScript）。

<script> 元素即可包含脚本语句，也可通过 src 属性指向外部脚本文件。

JavaScript 的常见用途是图像处理、表单验证和内容的动态更改。

如需选取 HTML 元素，JavaScript 最常用 document.getElementById() 方法。


这里的例子是在有“idname”的这个id的标签更改里面的内容

实例：

			
<p id="idname">发 生 啥 事 ！ 我 练 功 发 自 真 心 ！</p>
<script>
	document.getElementById('idname').innerHTML = "你好JavaScript";
</script>

PS:这个ID要写在script标签前才能检测到，而且无论这个标签写了什么内容，它都会被替换

<noscript> 标签

HTML <noscript> 标签定义了替代内容，这些内容将显示给在浏览器中禁用了脚本或浏览器不支持脚本的用户：

(讲白了就是当浏览器不能用 script 标签时 ，这个标签作为替代内容告诉读者用不了脚本)

</script>

HTML框架

通过使用框架，你可以在同一个浏览器窗口中显示不止一个页面。每份HTML文档称为一个框架，并且每个框架都独立于其他的框架。

直观的讲就像手机程序双/多开模式

使用框架的坏处：

  • 开发人员必须同时跟踪更多的HTML文档
  • 很难打印整张页面(估计是加载慢)

框架结构标签

框架结构标签（<frameset>）定义如何将窗口分割为框架

每个 frameset 定义了一系列行或列

rows/columns 的值规定了每行或每列占据屏幕的面积


框架标签

Frame 标签定义了放置在每个框架中的 HTML 文档。

在下面的这个例子中，我们设置了一个两列的框架集。第一列被设置为占据浏览器窗口的 25%。第二列被设置为占据浏览器窗口的 75%。
HTML 文档 "frame_a.html" 被置于第一个列中，而 HTML 文档 "frame_b.html" 被置于第二个列中：

					
<frameset cols="25%,75%">
	<frame src="frame_a.html">
	<frame src="frame_b.html">
 </frameset>

可以点击此处查看!(可以随意拖动之间的分割线)


这里是两行(第二行两列)的框架集：

					
<frameset rows="50%, 50%">

<frame src="frame_c.html">

<frameset cols="25%,75%">
	<frame src="frame_a.html">
	<frame src="frame_b.html">
 </frameset>

</frameset>

可点击此处查看！

PS: cols是设置列的属性，rows是设置行的属性

使用该标签的注意事项:

假如一个框架有可见边框，用户可以拖动边框来改变它的大小。为了避免这种情况发生，可以在 <frame> 标签中加入：noresize="noresize"。

为不支持框架的浏览器添加 <noframes> 标签。

不能将 <body></body> 标签与 <frameset></frameset> 标签同时使用！不过，假如你添加包含一段文本的 <noframes> 标签，就必须将这段文字嵌套于 <body></body> 标签内。

各种实例：

<noframes> :浏览器不支持框架时提示的替代信息：

					
<noframes>
	<body>
		哦豁:( <br />
		看来你所使用的服务器不支持框架呢！ <br />
		请换个浏览器试试吧！
	</body>
</noframes>

含有 noresize="noresize" 属性的框架结构(使用此属性你不能拖动分割线来改变网页结构)

					
<html>

<frameset cols="30%,*,25%">
  <frame src="frame_a.html" noresize="noresize" />
  <frame src="frame_b.html" />
  <frame src="frame_c.html" />
</frameset>

</html>

点击此处查看！

导航框架(以及跳转到指定的部分)

					
<frameset cols="120, *">
	<frame src="导航链接.html">
	<frame src="frame_a.html" name= "showframe">
</frameset>
